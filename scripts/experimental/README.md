# scripts/experimental/

Proof-of-concept and exploratory agent scripts. These are **not production code** — they're research artifacts from early architecture exploration.

| Script | Description |
|---|---|
| `autonomous_agent.py` | Autonomous LLM orchestration loop (Phase 4 prototype) |
| `mcp_agent.py` | MCP-based agent integration |
| `agent_router.py` | Multi-agent routing layer |
| `voice_input.py` | Voice I/O via Whisper (Phase 5 prototype) |

Stable/operational scripts live in `../` (parent `scripts/` folder).
